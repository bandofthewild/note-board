Note Board — Quick Tour

Get Going

  • Pick an Ensemble and a Note Set, then press Play.
  • Each turn the column pulses top‑to‑bottom (Get Ready, 4‑3‑2‑1), students say the note name, then optionally play it.
  • Click a colored column header to pin a ★ Focus Note — it gets drilled every other turn.

Use Your Own Music

  • Set Groove to Off and start your own track.
  • Tap the beat four times (or press the spacebar) to set the tempo. Four taps works best — too few or too many can throw off the reading.
  • If the class drifts, tap another four‑count along with the song and the board re‑syncs on the next downbeat.

Good To Know

  • The Instruments list is always shown — leave everyone checked, or uncheck instruments you don't need to keep the board uncluttered. A row shows two octaves when its members read in different registers.
  • Advanced Options holds play order, note‑name reveal, reference pitch, speed trainer, and concert‑pitch display.
  • Save Current stores a class setup by name, and Present goes fullscreen for the projector.

Note Selection

How notes are chosen. Random deals evenly from a shuffled bag (no immediate repeats). In order steps low-to-high through the set — good for introducing a new group of notes.

Helpers

Shows each line's own letter name (transposed per instrument) — but only on the last two beats of Say It, so students get a chance to figure it out first.

Sounds the actual concert pitch of each note as it appears, so students can match or tune to it. Best for small groups or personal practice.

Challenge & Display

Nudges the tempo up 2 bpm every turn for a build-up challenge. Works with the built-in grooves (not with the "Off" play-along mode).

Shows every line at concert pitch instead of each instrument's transposed reading — handy for talking about concert pitch across the whole room.

Click a colored header to pin a focus note (★) — it gets drilled every other turn while the rest of the set fills the gaps. A row shows both octaves when its members read in different registers (e.g. trombone + tuba), and notes past an instrument's range are hidden automatically.
